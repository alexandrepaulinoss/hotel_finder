## Hotel finder

Find the cheapest tax for a hotel (given a hotel and taxes list).

## Stack

Javascript

## Install the required modules

```
$ yarn install
```

## Run the tests

```
$ yarn test
```

## Run the app

```
$ node app.js
```

## Usage

```
help: for examples
```

```
quit: go back to terminal
```

```
Input example: Regular: 16Mar2009(mon), 17Mar2009(tues), 18Mar2009(wed)
```
